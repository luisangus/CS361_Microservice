This python script uses matplotlib to generate the barchart. In order to use it this module needs to be installed. Use the following command to install the module.

python -m pip install -U matplotlib

if your computer already has python 2.7 installed you will have to specify to call python 3.0 by doing the following:

python3 -m pip install -U matplotlib


If PIP is not installed in your computer, follow the next steps:
1. Download the get-pip.py file and store it in the same directory as python is installed.
2. Change the current path of the directory in the command line to the path of the directory where the above file exists.
3. Run the command given below:

python get-pip.py

and wait through the installation process.
